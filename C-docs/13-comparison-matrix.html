Capabilities

Built differently.

Six capabilities that make hanimo the most technically rigorous AI coding agent available.

#!
Hash-Anchored Editing

MD5 per-line hashing ensures edits are applied to exactly the right lines. The only tool that verifies edit correctness before applying.

# .hanimo.md edit_strategy: hash-anchored verify: md5 # hash: a3f8d91c ← verified
⇄
Super / Dev / Plan Modes

Three operating modes via Tab. Super for full autonomy. Dev for interactive pair-programming. Plan for structured multi-step execution.

# config.yaml modes: super: "full-autonomy" dev: "interactive" plan: "structured"
/▶
/auto Autonomous Mode

Up to 20 autonomous iterations. Reads, writes, tests, fixes, repeats. Stops and reports. Configurable iteration limit.

/auto refactor the auth module /auto --max-iter=10 \ add pagination to API
DB
SQLite Sessions

Every conversation persisted to SQLite. Save, load, fork, and full-text search. Resume exactly where you left off.

/session save auth-refactor-v2 /session fork auth-refactor-v2 /session search "pagination"
◎
3-Stage Compaction

Summarize at 80%, compress at 90%, evict at 95%. Never hit a context wall mid-task. Sessions stay coherent across thousands of tokens.

context: stages: 80%: summarize 90%: compress 95%: evict
MCP
MCP stdio + 14 Tools

Full Model Context Protocol via stdio. 14 built-in tools: file edit, shell, browser, code analysis. Ctrl+K command palette.

mcp_servers: postgres: cmd: mcp-postgres args: ["$DB_URL"]

Quick Start

Up and running in 60 seconds.

Choose your preferred install method.

# Download and install binary $ curl -fsSL https://hanimo.dev/install.sh | sh # Set API key for your preferred provider $ export ANTHROPIC_API_KEY=sk-ant-... # Or: OPENAI_API_KEY, GOOGLE_API_KEY, GROQ_API_KEY # Launch hanimo $ hanimo
# Tap the Homebrew cask $ brew tap flykimjiwon/hanimo $ brew install hanimo # Set API key $ export ANTHROPIC_API_KEY=sk-ant-... $ hanimo
# Requires Go 1.22+ $ git clone https://github.com/flykimjiwon/hanimo $ cd hanimo $ go build -o hanimo ./cmd/hanimo $ sudo mv hanimo /usr/local/bin/ $ export ANTHROPIC_API_KEY=sk-ant-... $ hanimo
