Unique capability

Hash-Anchored Editing. Zero hallucinated changes.

hanimo computes an MD5 hash for every line in your files before editing. Each edit is verified against its hash — guaranteeing changes land on exactly the right lines, every time.

  • MD5 hash computed per line before any edit session
  • Every proposed edit verified against stored hash before applying
  • Mismatches rejected — no silent wrong-line edits ever
  • First and only AI coding tool with line-level edit verification
.hanimo.md · project context yaml
# Hash-anchored editing config edit_strategy: hash-anchored verify: md5 strict: true # Generated hash manifest (auto) hashes: auth/middleware.go: 42: a3f8d91c4b22e8f0 43: 7c2e541a9d83f4b2 44: 1e9f6c3d8a4b72e5 # Edit verified before apply: # line 42 hash matches ✓ → apply # line 42 hash mismatch → reject

Autonomous operation

/auto mode. Up to 20 iterations of real work.

/auto delegates a complete task to hanimo. It reads files, writes edits, runs your tests, fixes errors, and reports back. Up to 20 iterations of genuine autonomous engineering.

  • Reads and understands your codebase context automatically
  • Writes edits, creates files, runs shell commands
  • Executes your test suite and fixes failures inline
  • Configurable iteration limit: --max-iter=N

Quick Start

Up and running in 60 seconds.

Three ways to install hanimo. Choose what works for you.

# Install binary (macOS, Linux, Windows/WSL) $ curl -fsSL https://hanimo.dev/install.sh | sh # Configure your provider $ export ANTHROPIC_API_KEY=sk-ant-... # Or any other: OPENAI_API_KEY, GOOGLE_API_KEY, GROQ_API_KEY # Start hanimo $ hanimo
# Tap and install via Homebrew $ brew tap flykimjiwon/hanimo $ brew install hanimo # Configure provider $ export ANTHROPIC_API_KEY=sk-ant-... $ hanimo
# Build from source (requires Go 1.22+) $ git clone https://github.com/flykimjiwon/hanimo $ cd hanimo $ go build -o hanimo ./cmd/hanimo $ sudo mv hanimo /usr/local/bin/ $ export ANTHROPIC_API_KEY=sk-ant-... $ hanimo
