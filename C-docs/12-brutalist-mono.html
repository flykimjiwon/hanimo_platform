01

CAPABILITIES

01

HASH-ANCHORED EDITING

MD5 per line. Edits verified before apply. Zero hallucinated changes. The only tool that guarantees edit correctness at the line level.

# .hanimo.md project config edit_strategy: hash-anchored verify: md5 strict: true # line 42: hash=a3f8d91c
02

THREE OPERATING MODES

Tab to switch: Super (full autonomy), Dev (interactive pair-programming), Plan (structured multi-step). Context preserved across mode switches.

# config.yaml modes: super: full-autonomy dev: interactive plan: structured default_mode: dev
03

/AUTO AUTONOMOUS MODE

Up to 20 iterations. Reads code. Writes edits. Runs tests. Fixes errors. Reports back. Like delegating to a senior engineer who never gets distracted.

# autonomous execution /auto add pagination to users API /auto --max-iter=10 \ refactor auth module
04

SQLITE SESSIONS

Every conversation persisted to SQLite. Save. Load. Fork. Full-text search across all sessions. Resume across days, branches, machines.

/session save auth-refactor-v2 /session fork auth-refactor-v2 /session search "pagination bug" /session load auth-refactor-v2
05

3-STAGE CONTEXT COMPACTION

Summarize at 80%. Compress at 90%. Evict at 95%. Never hit a context wall mid-task. Long sessions stay sharp and coherent automatically.

context: stages: 80%: summarize 90%: compress 95%: evict preserve: [system, tools]
06

MCP STDIO SUPPORT

Model Context Protocol via stdio. Connect databases, APIs, custom tools. One config line per server. 14 built-in tools + unlimited MCP extensibility.

mcp_servers: postgres: cmd: mcp-postgres args: ["$DATABASE_URL"] github: cmd: mcp-github

INSTALL IN 60 SECONDS

# 1. Download binary $ curl -fsSL https://hanimo.dev/install.sh | sh # 2. Set API key (pick your provider) $ export ANTHROPIC_API_KEY=sk-ant-... # or: OPENAI_API_KEY, GOOGLE_API_KEY, etc. # 3. Launch $ hanimo
# 1. Tap and install $ brew tap flykimjiwon/hanimo $ brew install hanimo # 2. Configure provider $ export ANTHROPIC_API_KEY=sk-ant-... # 3. Launch $ hanimo
# Requires Go 1.22+ $ git clone https://github.com/flykimjiwon/hanimo $ cd hanimo # Build $ go build -o hanimo ./cmd/hanimo # Install to PATH and run $ sudo mv hanimo /usr/local/bin/ $ hanimo
