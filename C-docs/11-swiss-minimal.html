01 / FEATURES

Built for serious engineering.

01 — Unique

Hash-Anchored Editing

Every line is MD5-hashed before editing. Edits are verified against the hash — eliminating hallucinated changes to the wrong lines.

# .hanimo.md — project context
edit_strategy: hash-anchored
verify: md5
strict: true
# Line 42 hash: a3f8d91c...
# Edit verified before apply
02 — Modes

Super / Dev / Plan

Three operating modes accessible via Tab. Super mode for complex autonomous tasks. Dev for interactive coding. Plan for structured multi-step execution.

# Tab to cycle modes
modes:
  super: "full autonomy"
  dev:   "interactive"
  plan:  "structured"
default: dev
03 — Autonomous

/auto Mode

Up to 20 iterations of autonomous action. Reads, writes, runs tests, fixes errors — then stops and reports. Like having a senior engineer run in the background.

# Autonomous execution
/auto add pagination to API

# With iteration limit
/auto --max-iter=10 \
  refactor auth module
04 — Context

Smart Compaction

3-stage context compaction prevents runaway token usage. Summarize → compress → evict. Your session stays sharp across long-running tasks.

context:
  strategy: "3-stage"
  stages:
    - summarize   # 80% fill
    - compress    # 90% fill
    - evict       # 95% fill
  preserve: ["system", "tools"]
05 — Sessions

SQLite Sessions

Every conversation is saved to SQLite. Save, load, fork, and search sessions. Resume exactly where you left off, across days or machines.

# Session management
/session save auth-refactor-v2
/session fork auth-refactor-v2
/session search "pagination"
/session load auth-refactor-v2
06 — MCP

MCP stdio Support

Full Model Context Protocol support via stdio. Connect to any MCP server — databases, APIs, custom tools — with a single config line.

# config.yaml — MCP setup
mcp_servers:
  postgres:
    cmd: "mcp-postgres"
    args: ["--url", "$DB"]
  github:
    cmd: "mcp-github"

04 / INSTALL

Running in under 60 seconds.

# Install latest binary
curl -fsSL https://hanimo.dev/install.sh | sh

# Or specify version
curl -fsSL https://hanimo.dev/install.sh | sh -s v0.9.0

# Set your provider
export ANTHROPIC_API_KEY=sk-ant-...

# Start hanimo
hanimo
# Tap the hanimo cask
brew tap flykimjiwon/hanimo

# Install
brew install hanimo

# Set your provider
export ANTHROPIC_API_KEY=sk-ant-...

# Start hanimo
hanimo
# Requires Go 1.22+
git clone https://github.com/flykimjiwon/hanimo
cd hanimo

# Build
go build -o hanimo ./cmd/hanimo

# Install to PATH
sudo mv hanimo /usr/local/bin/

# Start
hanimo
