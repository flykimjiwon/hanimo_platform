6가지 핵심 기능
01 ── EDIT ENGINE
해시 기반 편집
콘텐츠 해시로 정밀 편집. 오류 없는 수술적 수정.
02 ── PROVIDERS
14+ LLM 프로바이더
Ollama, OpenAI, Anthropic 등. 세션 중 전환 가능.
03 ── MEMORY
스마트 압축
컨텍스트 한계 시 자동 요약. 200K 토큰 이상 작업.
04 ── AGENT
/auto 자율 모드
작업 위임 후 자율 실행. 전체 감사 추적 제공.
05 ── PERSISTENCE
SQLite 세션
로컬 SQLite에 대화 저장. 데이터는 내 기기에.
06 ── PROTOCOL
MCP 지원
외부 도구·DB·API를 컨텍스트로. 포크 없이 확장.
