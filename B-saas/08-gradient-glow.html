/auto Autonomous Mode
Up to 20 self-directed iterations. Full loop: read → analyze → write → diagnose → repeat. No human in the loop required.
🧠
Smart 3-Stage Compaction
Intelligent context management keeps long sessions coherent. Compress old context, keep critical info sharp — never lose the thread.
🎯
3 Execution Modes
Super — full autonomy · Dev — interactive pair programming · Plan — think before act
💾
SQLite Sessions
All sessions persist locally in SQLite. Resume any conversation, full-text search history, export logs. Your data, your machine, always.
🔌
MCP Support
Full Model Context Protocol integration. Connect to any MCP server to extend hanimo's capabilities with external tools and data sources.

Single Go Binary
One binary, zero runtime dependencies. No Node, no Python, no Docker. Cross-platform: Linux, macOS, Windows. Ship anywhere instantly.

Get Started

Install in seconds.

$ curl -fsSL https://hanimo.dev/install | sh

# Installs the latest hanimo binary — Linux & macOS (arm64 + amd64)

$ brew install flykimjiwon/tap/hanimo

# Homebrew tap — auto-updates with brew upgrade

$ go install github.com/flykimjiwon/hanimo@latest

# Build from source — requires Go 1.21+

Then run hanimo in any project directory
