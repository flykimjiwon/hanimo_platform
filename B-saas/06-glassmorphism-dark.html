해시 기반 편집

Every file edit is anchored to a content hash. The AI verifies the exact file state before modifying — zero stale-state edits, zero silent corruption. Your codebase integrity is mathematically guaranteed.

/auto Mode

Fully autonomous execution. Up to 20 iterations. Reads, writes, runs diagnostics, self-corrects.

Smart Compaction

Get Started

Install in seconds.

$ curl -fsSL https://hanimo.dev/install | sh
# Downloads and installs the latest hanimo binary
# Supports Linux, macOS (arm64 + amd64)
$ brew install flykimjiwon/tap/hanimo
# Homebrew tap for macOS and Linux
$ go install github.com/flykimjiwon/hanimo@latest
# Requires Go 1.21+

Then run: hanimo in any project directory
