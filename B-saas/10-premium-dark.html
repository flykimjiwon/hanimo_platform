Capabilities

Built different. Everything by design.

01 · Unique Feature

Hash-Anchored Editing only hanimo

Every edit is anchored to a cryptographic content hash. The agent verifies exact file state before writing — stale-state corruption is mathematically impossible. No other AI coding tool does this.

→ read(file) # capture hash
→ verify(hash: a3f7d2b1) # confirm state
→ edit(anchored: true) # safe write

02

/auto Autonomous Mode

Up to 20 self-directed iterations. Reads, writes, runs diagnostics, self-corrects — no human in the loop. Full stack, fully autonomous.

03

Smart 3-Stage Compaction

Intelligent context management. Long sessions stay sharp — critical context preserved, verbose history compressed without losing the thread.

04

3 Execution Modes

Installation

Install in seconds.

Three install methods. All produce a single binary with zero runtime dependencies.

$curl -fsSL https://hanimo.dev/install | sh
# Downloads and installs the latest hanimo binary (Linux & macOS · arm64 + amd64)
$brew install flykimjiwon/tap/hanimo
# Homebrew tap — updates automatically with brew upgrade
$go install github.com/flykimjiwon/hanimo@latest
# Build from source · requires Go 1.21+

Then run hanimo in any project directory to start.
