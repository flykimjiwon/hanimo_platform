01
Capabilities
01

Core
Features

i

Hash-Anchored Editing Unique

Every file modification is cryptographically anchored to a content hash. The AI reads the hash, verifies the exact state of your file, then edits. Stale-state corruption is mathematically impossible. No other AI coding agent does this.

verify(hash: a3f7d2b1) → edit → re-verify → commit ✓
ii

/auto Autonomous Mode

Fully autonomous execution up to 20 self-directed iterations. hanimo reads, writes, runs diagnostics, self-corrects, and repeats — all without human intervention. Invoke with /auto [task description].

iii

Smart 3-Stage Compaction

Intelligent three-stage context compaction keeps long sessions sharp. Critical context is preserved; verbose history is compressed. Never lose the thread across multi-hour coding sessions.

iv

Three Execution Modes

Super — full autonomy with minimal confirmation. Dev — interactive pair programming mode. Plan — think and outline before acting.

v

SQLite Session Persistence

Every conversation is stored locally in SQLite. Resume any session, search full history, export logs, replay decisions. Your data never leaves your machine. Use /save and /remember to annotate sessions.

vi

MCP Protocol Support

Full Model Context Protocol integration. Connect hanimo to any MCP server — databases, APIs, custom tools, external knowledge sources. Extend capabilities without forking.

vii

Single Go Binary

Zero runtime dependencies. No Node.js, no Python, no Docker daemon. One binary. Works on Linux (amd64, arm64), macOS (arm64, amd64), and Windows. Distribute, containerize, embed — trivially.

viii

14+ LLM Providers

Truly model-agnostic. Ollama for local inference, OpenAI, Anthropic, Google Gemini, DeepSeek, Groq, Mistral, and seven more. Switch providers mid-session. No rewrite required.

Get
Started

Three ways to install. All result in a single binary with zero runtime dependencies.

$ curl -fsSL https://hanimo.dev/install | sh
# Installs the latest binary — Linux & macOS · arm64 & amd64
$ brew install flykimjiwon/tap/hanimo
# Tap-based install · auto-updates with brew upgrade
$ go install github.com/flykimjiwon/hanimo@latest
# Build from source · requires Go 1.21+
