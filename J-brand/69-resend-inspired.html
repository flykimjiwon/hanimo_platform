기능

필요한 것만.

불필요한 UI 없이 터미널에서 동작하는 여섯 가지 핵심 기능.

01
해시 앵커 편집

줄 번호 없이 콘텐츠 해시로 정밀 편집. 드리프트 없음.

02
14개 LLM 공급자

Ollama부터 OpenAI까지. 단일 인터페이스.

03
스마트 압축

컨텍스트 자동 관리. 비용 걱정 없는 긴 세션.

04
/auto 자율 모드

한 줄로 멀티스텝 자율 실행. 중간에 멈추지 않음.

05
SQLite 세션

로컬 SQLite에 저장. 클라우드 불필요. 완전 프라이빗.

06
MCP 지원

Model Context Protocol로 외부 도구 연결.
