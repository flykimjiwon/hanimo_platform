설치

30초 안에 시작.

$ curl -L https://hanimo.dev/install/hanimo-darwin-arm64 -o hanimo
$ chmod +x hanimo && mv hanimo /usr/local/bin/
$ hanimo
$ brew install hanimo
$ hanimo
$ git clone https://github.com/flykimjiwon/hanimo
$ cd hanimo && go build ./cmd/hanimo
$ ./hanimo
