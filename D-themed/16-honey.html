Features

Built for the terminal.
Designed for coders.

01 / editing
Hash-Anchored Editing

Every file edit is anchored by a content hash, preventing stale edits on files that have changed. No more overwritten work.

middleware/auth.go [#a3f2b1]
02 / providers
14+ LLM Providers

Ollama, OpenAI, Anthropic, Google, DeepSeek, Groq, Mistral, and more. Switch providers with a single flag.

--provider anthropic --model claude-3-5-sonnet
03 / memory
Smart Compaction

Intelligent context window management. Automatically summarizes and compacts long sessions, keeping the most relevant context.

Context: 68% · Compacting in 8k tokens
04 / autonomy
/auto Autonomous Mode

Full autonomous execution. hanimo plans, reads, edits, runs tests, and iterates — all without intervention. Just describe the goal.

/auto add pagination to the user list API
05 / sessions
SQLite Sessions

Every session is persisted locally in SQLite. Resume any past conversation exactly where you left off with full context.

/save auth-refactor · /remember context
06 / tools
MCP Support

Full Model Context Protocol support. Connect databases, APIs, file systems, and custom tools to any compatible model.

MCP: filesystem · postgres · github
